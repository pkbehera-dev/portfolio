# Portfolio

A personal portfolio website showcasing skills, projects, and professional experience.

## Overview

This is a responsive personal portfolio website built with HTML, CSS, and JavaScript. It serves as a digital presence to showcase professional work, skills, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Clean UI**: Modern and professional design aesthetic
- **Interactive Elements**: Smooth animations and transitions
- **Project Showcase**: Display of completed projects and work samples
- **Contact Section**: Easy way to get in touch

## Project Structure

```
portfolio/
├── index.html              # Main HTML file
├── assets/
│   ├── style.css          # Stylesheet
│   ├── script.js          # JavaScript functionality
│   ├── me.jpeg            # Profile photo
│   └── PRADYUMNA KUMAR BEHERA.pdf  # Resume/CV
```

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: A local web server for development

### Running the Website

1. **Direct Open**: Simply double-click `index.html` to open it in your default web browser

2. **Using a Local Server** (recommended for development):

   Using Python:

   ```bash
   python -m http.server 8000
   ```

   Then open `http://localhost:8000` in your browser

   Using Node.js (if installed):

   ```bash
   npx serve
   ```

## Customization

### Updating Profile Information

Edit `index.html` to update your name, bio, and professional information.

### Modifying Styles

Edit `assets/style.css` to customize colors, fonts, and layout.

### Adding/Modifying Projects

Edit `index.html` to add or modify project entries in the portfolio section.

### Updating Resume

Replace `assets/PRADYUMNA KUMAR BEHERA.pdf` with your updated resume (keep the same filename or update the link in HTML).

### Changing Profile Photo

Replace `assets/me.jpeg` with your new profile photo (keep the same filename or update the link in HTML).
